# Voice Samples

## `raven_chris_goringe_librivox_pd.wav`

Short public-domain voice sample derived from Chris Goringe's LibriVox reading
of Edgar Allan Poe's `The Raven`.

Regenerate it with:

```sh
bash tools/fetch-raven-voice-sample.sh
```

The adjacent TOML file records the source URL, trim point, conversion settings,
and license notes.
